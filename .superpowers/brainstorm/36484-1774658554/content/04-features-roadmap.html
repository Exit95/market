Features, Roadmap & Obsidian-Hub

Was gebaut wird, in welcher Reihenfolge, und wie das Obsidian-Vault alles zusammenhalt.

Phase 1 — Design-System & Homepage (Fundament)
Design-System
  • Neue Farbpalette (Trust Blue, Safe Green, Navy)
  • Typografie: Inter/System Stack, kein Serif
  • Tailwind Config komplett neu
  • Kern-Komponenten: Buttons, Cards, Badges, Forms
  • global.css entrumpeln: Grain, Glass weg
Homepage
  • Hero: Trust-First Messaging, Suchleiste
  • Trust Bar: 3 Saulen
  • Kategorie-Pills (clean, wenig Icons)
  • Listing-Grid mit neuen Cards
  • "So funktioniert's" 3-Schritte
  • Risk-Reversal Block + finaler CTA
Phase 2 — Kern-Seiten Redesign
Listing-Seite
  • Bildergalerie gross + Thumbnails
  • Risk-Reversal Box am CTA
  • Seller-Card mit Trust-Score
  • Transparente Preise/Gebuhren
  • Breadcrumbs + Structured Data
Auth & Profil
  • Login/Register: Light-Theme
  • Multi-Step Register (Micro-Commitment)
  • Profil: Trust-Score prominent
  • Verifizierungs-Fortschritt sichtbar
  • Offentliches Profil mit Deal-History
Inserat erstellen
  • 4-Step Wizard beibehalten
  • Fortschrittsbalken (Zeigarnik)
  • Drag & Drop Bildupload
  • Bessere Formular-Validierung
  • Vorschau vor Veroffentlichung
Phase 3 — Deal-Flow & Checkout
Checkout & Payment
  • Checkout-Seite: clean, Stripe-Einbettung
  • Treuhand-Erklarung im Checkout
  • Fortschrittsanzeige: Bezahlen → Versand → Bestatigung
  • Bestatigungsseite mit nachsten Schritten
Deal-Management
  • Deal-Detail: Status-Timeline statt Textliste
  • Dashboard: Ubersichtliche Tabelle/Cards
  • Versand-Tracking Integration
  • Streitfall-Flow mit klaren Schritten
Phase 4 — Neue Features (was Kleinanzeigen nicht hat)
Bewertungssystem
  • Bewertung nach abgeschlossenem Deal
  • Sterne + Freitext
  • Bewertungen offentlich auf Profil
  • Vault: "4.7/5 glaubwurdiger als 5.0"
Gamification (leicht)
  • Trust-Level: Neu → Verifiziert → Trusted → Elite
  • Fortschrittsbalken: "Noch 2 Deals bis Trusted"
  • Profil-Vollstandigkeit anzeigen
  • Vault: Hook-Modell, Zeigarnik-Effekt
Suche & Entdecken
  • Filter: Preis, Zustand, Entfernung, Treuhand
  • Sortierung: Relevanz, Preis, Neueste
  • Algolia Full-Text-Suche aktivieren
  • Gespeicherte Suchen / Benachrichtigungen
Phase 5 — Performance, SEO & Wachstum
Performance
  • LCP < 2.5s (AVIF + WebP Bilder)
  • INP < 200ms (lazy JS, defer non-critical)
  • CLS < 0.1 (explicit image dimensions)
  • Fonts: self-hosted, font-display: optional
  • Vault: "Jede 100ms = +1% Conversions"
SEO & Structured Data
  • Schema: Product, BreadcrumbList, WebSite
  • Open Graph Tags fur Social Sharing
  • Sitemap + robots.txt optimiert
  • WCAG 2.2 AA Konformitat
  • Vault: "Schema erhoht Rich-Snippet-Chancen 40-60%"
