Design-Entscheidungen & Vault-Begrundungen
Risk Reversal direkt am CTA

Vault Trust 8.3: "Je radikaler die Garantie, desto starker der Effekt." Die blaue Info-Box direkt unter dem Kaufen-Button eliminiert die letzte Kaufangst im entscheidenden Moment.

Trust-Score beim Verkaufer

Vault Social Proof 3.1 + 3.3: "Spezifische Zahlen statt runder: 94 statt 'gut'." Foto + Name + Score = hochste Glaubwurdigkeit. Vault: "Testimonials mit Foto erhohen Glaubwurdigkeit um 35%"

Preis transparent, keine versteckten Kosten

Vault CRO-Killer #7 (Friction): "Versteckte Kosten erst im Checkout = Conversion-Killer." Treuhand-Gebuhr und Versand direkt sichtbar. Transparenz = Vertrauen.

Breadcrumb + klare Hierarchie

Vault CRO UX: "Findet der Nutzer in 3 Sek was er sucht?" Breadcrumbs + Structured Data (SEO 2026: BreadcrumbList Schema). Doppelter Nutzen: UX + SEO.

Kern-Komponenten fur alle Seiten
Listing Card
Bild + Badges + Titel + Ort/Zeit + Preis. Weisser Hintergrund, subtiler Border, 12px Radius. Treuhand/Verifiziert-Badges nur als farbige Text-Labels, keine Icons.
Trust-Badge
Grun (#E8F5E9 bg, #22A06B text) fur Treuhand. Blau (#E3F2FD bg, #1B65A6 text) fur Verifiziert. Kein Icon, nur farbiger Text-Chip. Dezent aber klar.
Seller Card
Avatar (Initialen) + Name + Mitglied-seit + Deal-Count + Trust-Score. Kompakt, glaubwurdig. Vault: "Foto + Name + Beruf = hochste Glaubwurdigkeit."
CTA Buttons
Primary: Trust Blue, 16px font, 10px radius, grosszugiges Padding. Secondary: Weiss mit Border. Vault: "Abgerundete Ecken wirken freundlicher und klickbarer."
Info Box
Hellblauer Hintergrund (#F0F7FF), kein Border, 10px Radius. Fur Risk-Reversal-Texte, Hinweise, Erklarungen. Ruhig, nicht alarmierend.
Form Fields
Weisser Hintergrund, 1px #E5E7EB Border, 8px Radius, 16px Font. Focus: Blue Border + leichter Blue Shadow. Vault CRO: "Klare, hilfreiche Fehlermeldungen."
