# Ehren-Deal – Development Branch

Dieser Branch dient als Integrationsbranch für alle Feature-Entwicklungen.
Neue Features werden hier zusammengeführt, bevor sie in einen Release-Branch fließen.

## Workflow
1. Feature-Branch von `develop` abzweigen
2. Feature implementieren & testen
3. Pull Request → `develop`
4. Release-Branch von `develop` erstellen
5. Nach Abnahme: Merge in `main` + Tag
