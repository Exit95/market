# User-Verifizierung

## Ziel
Vertrauenswürdigkeit der Nutzer durch mehrstufige Verifizierung stärken.

## Geplante Features
- E-Mail Verifizierung (bereits vorhanden)
- Telefonnummer Verifizierung (SMS-Code)
- Ausweis-Verifizierung (ID-Upload + KI-Check)
- Trust-Level Anzeige im Profil
- Verifizierungs-Badge auf Inseraten
