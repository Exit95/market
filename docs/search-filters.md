# Erweiterte Suchfilter

## Ziel
Bessere Auffindbarkeit von Inseraten durch erweiterte Filter.

## Geplante Features
- Umkreissuche (PLZ + Radius)
- Zustandsfilter (Neu, Gebraucht, Defekt)
- Preissortierung
- Treuhand-Filter (nur sichere Deals)
- Volltextsuche mit Relevanz-Ranking
